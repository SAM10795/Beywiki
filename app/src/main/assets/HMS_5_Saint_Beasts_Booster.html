Contents of HMS 5 Saint Beasts Booster

Note: There is no prize Beyblade in this booster, because it is a compilation of the 5 Saint Beast Beyblades, save for the secret prize, the Ice Blue Wolborg MS. It is speculated that the "Proto Grey" color refers to the color of prototype Beyblades.

  • Wolborg MS (Secret Beyblade)(Ice Blue)
  • Draciel MS (Proto Grey)
  • Driger MS (Proto Grey)
  • Dranzer MS (Proto Grey)
  • Dragoon MS UV (Proto Grey)
  • Gaia Dragoon MS (Proto Grey)
