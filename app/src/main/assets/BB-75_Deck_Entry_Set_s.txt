BB-75 Deck Entry Set - Beywiki, the Beyblade Encyclopedia - Presented by the World Beyblade Organization BB-75 Deck Entry Set From Beywiki, the Beyblade Encyclopedia - Presented by the World Beyblade Organization The BB-75 Deck Entry Set was a Takara-Tomy Beyblade set that conisisted of Rock Orso ED145D, Galaxy Pegasis W105R2F, and Earth Virgo T125ES.
